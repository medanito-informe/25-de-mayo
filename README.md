# El Medanito, explicado con datos

Pagina estatica para publicar un resumen ciudadano del Informe Final Ambiental del area hidrocarburifera El Medanito.

## Publicar con GitHub Pages

1. Subir estos archivos al repositorio.
2. Entrar a `Settings` -> `Pages`.
3. En `Build and deployment`, elegir `Deploy from a branch`.
4. Seleccionar branch `main` y carpeta `/ (root)`.
5. Guardar. La pagina quedara disponible en la URL de GitHub Pages del repositorio.

## Contenido

- `index.html`: contenido principal.
- `styles.css`: estilos responsivos e impresion.
- `script.js`: soporte progresivo para componentes interactivos si se agregan.

La pagina evita nombres personales o politicos y se concentra en datos procesados del informe ambiental.
